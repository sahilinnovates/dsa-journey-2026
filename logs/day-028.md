### 1) Problem: Find the City with the Smallest Number of Neighbours

**short description of problem**
Given a weighted undirected graph of cities and a distance threshold, determine the city that can reach the smallest number of other cities within the given threshold distance. If multiple cities satisfy the condition, return the city with the greatest index.

**Approach:**

* Initialize an adjacency matrix and set large values (1e9) for unreachable pairs.
* Apply Floyd-Warshall algorithm to compute shortest distances between all pairs of cities.
* For each city, count how many cities are reachable within the threshold distance.
* Track the city having the minimum reachable count, preferring the greatest index in case of ties.

**Time Complexity:** `O(V^3)`
**Space Complexity:** `O(V^2)`

**Code:**

```cpp
class Solution {
public:
	void floyd_warshall(int n, vector<vector<int>>& mat){
		for(int via=0; via<n; via++){
			for(int i=0; i<n; i++){
				for(int j=0; j<n; j++){
					mat[i][j]=min(mat[i][j],mat[i][via]+mat[via][j]);
				}
			}
		}
	}
	int findCity(int n, int m, vector<vector<int>>& edges,
	             int distanceThreshold) {
		vector<vector<int>> mat(n,vector<int>(n,1e9));
		for(int i=0; i<n; i++){
			mat[i][i]=0;
		}
		for(auto it: edges){
			mat[it[0]][it[1]]=it[2];
			mat[it[1]][it[0]]=it[2];
		}
		floyd_warshall(n,mat);
		int minCities = n+1;
		int city=-1;
		for(int i=0; i<n; i++){
			int cnt=0;
			for(int j=0; j<n; j++){
				if(mat[i][j]<=distanceThreshold) cnt++;
			}
			if(cnt<=minCities){
				minCities=cnt;
				city=i;
			}
		}
		return city;
	}
};
```

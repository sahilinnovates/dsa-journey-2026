### Problem 1: Dijkstra's Algorithm (Using Priority Queue)

**Problem Statement**
Given a source node `S`, find the shortest distance from `S` to all other vertices in a weighted graph with non-negative weights.
If a vertex is unreachable, its distance should remain `10^9`.

---

### Approach

* Initialized distance array with `1e9`.
* Used a **Min Heap (Priority Queue)** storing `{distance, node}`.
* Set `dist[S] = 0` and pushed `{0, S}` into heap.
* While heap is not empty:

  * Extract node with minimum distance.
  * For each adjacent node:

    * Relax the edge if a shorter path is found.
    * Push updated distance into heap.

---

### Time Complexity

* **O(E log V)**

### Space Complexity

* **O(V)**
  For distance array and priority queue.

---

### Code

```cpp
class Solution{
public:
    vector<int> dijkstra(int V, vector<vector<int>> adj[], int S) {
        vector<int> dist(V,1e9);
        priority_queue<pair<int,int>,vector<pair<int,int>>,greater<pair<int,int>>> pq;
        pq.push({0,S});
        dist[S]=0;
        while(!pq.empty()){
            int d = pq.top().first;
            int nd = pq.top().second;
            pq.pop();
            for(auto it: adj[nd]){
                if(d+it[1]<dist[it[0]]){
                    dist[it[0]]=d+it[1];
                    pq.push({dist[it[0]],it[0]});
                }
            }
        }
        return dist;
    }
};
```

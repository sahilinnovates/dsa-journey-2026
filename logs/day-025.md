### 1) Problem: Number of Ways to Arrive at Destination

**short description of problem**
Given a weighted undirected graph where each edge represents travel time, determine the number of distinct ways to reach from node 0 to node n-1 using the minimum possible time. Return the result modulo 1e9+7.

**Approach:**

* Build an adjacency list for the graph storing (neighbor, time).
* Apply Dijkstra’s algorithm to compute the shortest distance from source node 0.
* Maintain a ways array to count the number of shortest paths to each node.
* If a new shortest distance is found, update both distance and ways; if an equal distance is found, add the number of ways.

**Time Complexity:** `O(E log V)`
**Space Complexity:** `O(V + E)`

**Code:**

```cpp
class Solution{
public:
    const int mod = 1e9+7;
    int countPaths(int n, vector<vector<int>> &roads) {
        vector<pair<int,int>> adj[n];
        int r = roads.size();
        for(int i=0; i<r; i++){
            adj[roads[i][0]].push_back({roads[i][1],roads[i][2]});
            adj[roads[i][1]].push_back({roads[i][0],roads[i][2]});
        }

        vector<int> ways(n,0);
        vector<long long> dist(n,1e18);
        priority_queue<pair<long long,int>,vector<pair<long long,int>>,greater<pair<long long,int>>> pq;
        pq.push({0,0});
        dist[0]=0;
        ways[0]=1;
        while(!pq.empty()){
            int d = pq.top().first;
            int nd = pq.top().second;
            pq.pop();
            for(auto it: adj[nd]){
                int node = it.first;
                int t = it.second;
                if(dist[node]==d+t){
                    ways[node]=(ways[node]+ways[nd])%mod;
                } else if(dist[node]>d+t){
                    dist[node]=d+t;
                    pq.push({dist[node],node});
                    ways[node]=ways[nd];
                }
            }
        }
        return ways[n-1];
    }
};
```

### 1) Problem: Check for Balanced Binary Tree

**Given the root of a binary tree, determine whether the tree is height-balanced. A binary tree is balanced if the height difference between the left and right subtree of every node is not more than 1.**

**Approach:**

* Use a recursive helper function to compute the height of subtrees.
* If any subtree is already unbalanced, propagate `-1` upwards.
* If the absolute difference between left and right subtree heights is greater than 1, mark it as unbalanced.
* Otherwise return the height of the current node.

**Time Complexity:** `O(N)`
**Space Complexity:** `O(H)`

**Code:**

```cpp
class Solution{
public:
    int check(TreeNode* nd){
        if(nd==NULL) return 0;
        int lh=check(nd->left);
        int rh=check(nd->right);
        if(lh==-1 || rh==-1) return -1;
        if(abs(lh-rh)>1) return -1;
        return 1+max(lh,rh);
    }
    bool isBalanced(TreeNode *root){
    	if(check(root)==-1) return false;
        return true;
    }
};
```

### 2) Problem: Diameter of Binary Tree

**The diameter of a binary tree is the length of the longest path between any two nodes in the tree. This path may or may not pass through the root.**

**Approach:**

* Use DFS to compute the height of left and right subtrees.
* For every node, the possible diameter passing through it is `leftHeight + rightHeight`.
* Maintain a global variable to store the maximum diameter encountered.

**Time Complexity:** `O(N)`
**Space Complexity:** `O(H)`

**Code:**

```cpp
class Solution {
public:
    int maxd=0;
    int check(TreeNode* nd){
        if(nd==NULL) return 0;
        int lh = check(nd->left);
        int rh = check(nd->right);
        maxd=max(maxd,lh+rh);
        return 1+max(lh,rh);
    }
    int diameterOfBinaryTree(TreeNode* root) {
        check(root);
        return maxd;
    }
};
```

### 3) Problem: Check for Symmetrical Binary Trees

**Given the root of a binary tree, determine whether the tree is symmetric around its center (mirror of itself).**

**Approach:**

* Compare the left and right subtrees recursively.
* Two nodes are symmetric if their values are equal and their children are mirror images.
* Check `(left.left, right.right)` and `(left.right, right.left)` recursively.

**Time Complexity:** `O(N)`
**Space Complexity:** `O(H)`

**Code:**

```cpp
class Solution {
public:
    bool check(TreeNode* nd1, TreeNode* nd2){
        if(nd1==NULL && nd2==NULL) return true;
        if(nd1==NULL || nd2==NULL) return false;
        if(nd1->data != nd2->data) return false;
        return check(nd1->left,nd2->right)&&check(nd1->right,nd2->left);
    }
    bool isSymmetric(TreeNode* root) {
        return check(root->left,root->right);
    }
};
```

### 1) Problem: Zig Zag Traversal

**Given the root of a binary tree, return the zigzag level order traversal of its nodes' values (left to right, then right to left, alternating each level).**
**Approach:**

* Perform standard **level-order traversal (BFS)** using a queue.
* For each level, store node values in a vector.
* Reverse the level vector on alternate levels using a boolean flag.

**Time Complexity:** `O(n)`
**Space Complexity:** `O(n)`

**Code:**

```cpp
vector<vector<int>> zigzagLevelOrder(TreeNode* root) {
    if(root==NULL) return {};
    vector<vector<int>> ans;
    queue<TreeNode*> q;
    bool rev = false;
    q.push(root);
    while(!q.empty()){
        int s = q.size();
        vector<int> v;
        for(int i=0; i<s; i++){
            TreeNode* nd = q.front();
            q.pop();
            v.push_back(nd->val);
            if(nd->left)q.push(nd->left);
            if(nd->right)q.push(nd->right);
        }
        if(rev) reverse(v.begin(),v.end());
        ans.push_back(v);
        rev = (!rev);
    }
    return ans;
}
```

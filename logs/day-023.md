### 1) Problem: Cheapest Flight Within K Stops

**short description of problem**
There are n cities connected by flights where each flight has a cost. Given source, destination, and an integer k, find the minimum cost to reach the destination from the source with at most k stops. If not possible, return -1.

**Approach:**

* Build adjacency list from the flights input storing (neighbor, cost).
* Use BFS-like traversal with a queue storing (stops, node, current cost).
* Relax edges only if stops are within limit and a cheaper cost is found.
* Maintain a distance array to track minimum cost to each node.

**Time Complexity:** `O(E)`
**Space Complexity:** `O(V + E)`

**Code:**

```cpp
class Solution {
public:
    int findCheapestPrice(int n, vector<vector<int>>& flights, int src, int dst, int k) {
        vector<vector<pair<int,int>>> adj(n);
        for(auto v: flights){
            adj[v[0]].push_back({v[1],v[2]});
        }
        vector<int> dist(n,1e9);
        queue<vector<int>> q;
        q.push({0,src,0});
        dist[src]=0;
        while(!q.empty()){
            int s = q.front()[0];
            int nd = q.front()[1];
            int d = q.front()[2];
            q.pop();
            for(auto it: adj[nd]){
                if(s+1<=k+1 && dist[it.first]>d+it.second){
                    dist[it.first]=d+it.second;
                    q.push({s+1,it.first,dist[it.first]});
                }
            }
        }
        if(dist[dst]==1e9) return -1;
        return dist[dst];
    }
};
```

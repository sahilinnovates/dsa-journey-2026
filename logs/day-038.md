### 1) Problem: Critical Connections in a Network

**Given an undirected network of servers, a critical connection (bridge) is an edge whose removal increases the number of connected components. The task is to find all such critical connections in the network.**

**Approach:**

* Use Tarjan’s Algorithm with DFS to track discovery time (`tin`) and the lowest reachable time (`low`).
* For each node, explore its neighbors while ignoring the parent edge.
* After visiting a neighbor, update the `low` value of the current node.
* If `low[neighbor] > tin[current]`, the edge between them is a bridge (critical connection).

**Time Complexity:** `O(V + E)`
**Space Complexity:** `O(V + E)`

**Code:**

```cpp
class Solution {
public:
    int timer=1;
    void dfs(int nd, int par, vector<int> adj[], vector<int>& vis, vector<int>& tin, vector<int>& low, vector<vector<int>>& ans){
        vis[nd]=1;
        tin[nd]=low[nd]=timer;
        timer++;
        for(auto it: adj[nd]){
            if(it==par) continue;
            if(!vis[it]) {
                dfs(it,nd,adj,vis,tin,low,ans);
                low[nd]=min(low[nd],low[it]);
                if(low[it]>tin[nd]){
                    ans.push_back({it,nd});
                }
            } else {
                low[nd]=min(low[nd],low[it]);
            }
        }
    };
    vector<vector<int>> criticalConnections(int n, vector<vector<int>>& connections) {
        vector<vector<int>> ans;
        vector<int> tin(n);
        vector<int> low(n);
        vector<int> adj[n];
        vector<int> vis(n,0);
        for(auto it: connections){
            adj[it[0]].push_back(it[1]);
            adj[it[1]].push_back(it[0]);
        }
        for(int i=0; i<n; i++){
            if(!vis[i]) dfs(i,-1,adj,vis,tin,low,ans);
        }
        return ans;
    }
};
```

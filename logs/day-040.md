### 1) Problem: Level Order Traversal of a Binary Tree

**Given the root of a binary tree, return the level order traversal of its nodes' values (i.e., from left to right, level by level).**

**Approach:**

* Push the root node into the queue and process nodes level by level.
* For each level, record the number of nodes currently in the queue and process them.
* Add their children to the queue for the next level and store the level values.

**Time Complexity:** `O(N)`
**Space Complexity:** `O(N)`

**Code:**

```cpp
class Solution {
public:
    vector<vector<int> > levelOrder(TreeNode* root) {
        if(root==NULL) return {};
        vector<vector<int>> ans;
        queue<TreeNode*> q;
        q.push(root);
        while(!q.empty()){
            int s = q.size();
            vector<int> v;
            for(int i=0; i<s; i++){
                TreeNode* nd = q.front();
                q.pop();
                v.push_back(nd->data);
                if(nd->left!=NULL) q.push(nd->left);
                if(nd->right!=NULL) q.push(nd->right);
            }
            ans.push_back(v);           
        }
        return ans;
    }
};
```

### 2) Problem: Maximum Depth of Binary Tree

**Given the root of a binary tree, find the maximum depth (height) of the tree. The depth is the number of nodes along the longest path from the root down to the farthest leaf node.**

**Approach:**

* Use recursion to compute the depth of left and right subtrees.
* The depth of the current node is `1 + max(leftDepth, rightDepth)`.
* Base case: if the node is `NULL`, return `0`.

**Time Complexity:** `O(N)`
**Space Complexity:** `O(H)`

**Code:**

```cpp
class Solution {
public:
    int maxDepth(TreeNode* root) {
        if(root==NULL) return 0;
        return 1+max(maxDepth(root->left),maxDepth(root->right));
    }
};
```

### 3) Problem: Check if Two Trees are Identical

**Given the roots of two binary trees, determine whether they are structurally identical and have the same node values.**

**Approach:**

* Recursively compare both trees simultaneously.
* If both nodes are `NULL`, they are identical.
* If only one node is `NULL` or values differ, trees are not identical.
* Recursively check the left and right subtrees.

**Time Complexity:** `O(N)`
**Space Complexity:** `O(H)`

**Code:**

```cpp
class Solution {
public:
    bool isSameTree(TreeNode* p, TreeNode* q) {
        if(p==NULL && q==NULL) return true;
        if(p==NULL || q==NULL) return false;
        if(p->data!=q->data) return false;
        return (isSameTree(p->left,q->left) && isSameTree(p->right,q->right));
    }
};
```

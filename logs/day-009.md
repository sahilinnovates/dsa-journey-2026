### 1) Problem: Detect a Cycle in a Directed Graph (Kahn’s Algorithm)

**Determine whether a directed graph contains a cycle using BFS-based topological sorting.**
**Approach:**

* Compute the indegree of each node.
* Push all nodes with indegree `0` into a queue.
* Perform **Kahn’s Algorithm** (BFS) to generate a topological order.
* If the topological order does not contain all `N` nodes, a cycle exists in the graph.

**Time Complexity:** `O(V + E)`
**Space Complexity:** `O(V)`

**Code:**

```cpp
class Solution{
public:
    vector<int> topoSort(int V, vector<int> adj[]){
        vector<int> indeg(V,0);
        queue<int> q;
        vector<int> ans;
        for(int i=0; i<V; i++){
            for(int j: adj[i]){
                indeg[j]++;
            }
        }
        for(int i=0; i<V; i++){
            if(indeg[i]==0) q.push(i);
        }
        while(!q.empty()){
            int nd = q.front();
            q.pop();
            ans.push_back(nd);
            for(int node: adj[nd]){
                if(--indeg[node]==0) q.push(node);
            }
        }
        return ans;
    }
    bool isCyclic(int N, vector<int> adj[]) {
        vector<int> topo = topoSort(N,adj);
        if(topo.size()!=N) return true;
        return false;
    }
};
```

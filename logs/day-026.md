### 1) Problem: Bellman Ford Algorithm
Given a directed weighted graph, find the shortest distance from a source vertex to all other vertices. If a vertex is unreachable, mark its distance as 1e9. If the graph contains a negative weight cycle, return -1.

**Approach:**

* Initialize all distances as infinity (1e9) except the source node.
* Relax all edges V-1 times to compute shortest paths.
* Perform one additional relaxation pass to detect negative cycles.
* If any edge still relaxes, a negative cycle exists.

**Time Complexity:** `O(V * E)`
**Space Complexity:** `O(V)`

**Code:**

```cpp
class Solution {
public:
	vector<int> bellman_ford(int V, vector<vector<int>>& edges, int S) {
		vector<int> dist(V,1e9);
		dist[S]=0;
		for(int i=0; i<V-1; i++){
			for(auto it: edges){
				int u = it[0];
				int v = it[1];
				int d = it[2];
				if(dist[u]!=1e9 && dist[v]>dist[u]+d){
					dist[v]=dist[u]+d;
				}
			}
		}
		for(auto it: edges){
			int u = it[0];
			int v = it[1];
			int d = it[2];
			if(dist[u]!=1e9 && dist[v]>dist[u]+d){
				return {-1};
			}
		}
		return dist;
	}
};
```

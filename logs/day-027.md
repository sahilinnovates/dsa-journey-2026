### 1) Problem: Floyd Warshall Algorithm
Given a directed weighted graph represented as an adjacency matrix, compute the shortest distance between every pair of vertices. If there is no direct edge between two vertices, the value is given as -1.

**Approach:**

* Use the Floyd-Warshall algorithm considering each node as an intermediate (via) node.
* Update matrix[i][j] if a shorter path exists through the intermediate node.
* Skip transitions where intermediate paths do not exist (-1 values).
* Modify the matrix in-place to store all-pairs shortest paths.

**Time Complexity:** `O(V^3)`
**Space Complexity:** `O(1)`

**Code:**

```cpp
class Solution {
public:
	void shortestDistance(vector<vector<int>>&matrix) {
		int n = matrix.size();
		for(int via=0; via<n; via++){
			for(int i=0; i<n; i++){
				for(int j=0; j<n; j++){
					if(matrix[i][via]==-1 || matrix[via][j]==-1) continue;
					if(matrix[i][j]==-1 || matrix[i][j]>matrix[i][via]+matrix[via][j]){
						matrix[i][j]=matrix[i][via]+matrix[via][j];
					}
				}
			}
		}
	}
};
```

### 1) Problem: Articulation Point in Graph

**Given an undirected graph, find all vertices whose removal (along with their edges) increases the number of connected components. These vertices are known as articulation points. The graph may be initially disconnected and vertices are numbered from 0 to V-1.**

**Approach:**

* Use DFS with discovery time (`tin`) and lowest reachable time (`low`) to detect articulation points.
* For a non-root node `u`, if `low[v] >= tin[u]` for any child `v`, then `u` is an articulation point.
* For a root node, if it has more than one DFS child, it is an articulation point.
* Maintain a marker array to track articulation points and return them in ascending order.

**Time Complexity:** `O(V + E)`
**Space Complexity:** `O(V + E)`

**Code:**

```cpp
class Solution {
public:
    int timer=0;
    void dfs(int nd, int par, vector<int> adj[], vector<int>& vis, vector<int>& low, vector<int>& tin, vector<int>& mark){
        vis[nd]=1;
        tin[nd]=low[nd]=timer;
        timer++;
        int child=0;
        for(auto it: adj[nd]){
            if(it==par) continue;
            if(!vis[it]){
                dfs(it,nd,adj,vis,low,tin,mark);
                low[nd]=min(low[nd], low[it]);
                if(low[it]>=tin[nd] && par!=-1){
                    mark[nd]=1;
                }
                child++;
            } else {
                low[nd]=min(low[nd],tin[it]);
            }
        }
        if(par==-1 && child>1) mark[nd]=1;
    }
    vector<int> articulationPoints(int n, vector<int>adj[]) {
        vector<int> vis(n,0);
        vector<int> low(n);
        vector<int> tin(n);
        vector<int> mark(n,0);
        vector<int> ans;
        for(int i=0; i<n; i++){
            if(!vis[i]) dfs(i,-1,adj,vis,low,tin,mark);
        }
        for(int i=0; i<n; i++){
            if(mark[i]==1) ans.push_back(i);
        }
        if(ans.size()==0) return {-1};
        return ans;
    }
};
```

### 1) Problem: Maximum Path Sum

**Given the root of a binary tree, return the maximum path sum. A path is any sequence of nodes connected by edges where each node appears at most once, and it does not necessarily need to pass through the root.**

**Approach:**

* Use DFS to compute the maximum contribution from left and right subtrees.
* Ignore negative contributions by taking `max(0, subtreeSum)`.
* For each node, calculate the possible maximum path passing through it: `node->data + left + right`.
* Maintain a global variable to store the overall maximum path sum.

**Time Complexity:** `O(N)`
**Space Complexity:** `O(H)`

**Code:**

```cpp
class Solution {	
public:
    int maxp=INT_MIN;
    int check(TreeNode* nd){
        if(nd==NULL) return 0;
        int lp = max(0,check(nd->left));
        int rp = max(0,check(nd->right));
        maxp = max(maxp,nd->data+lp+rp);
        return nd->data+max(lp,rp);
    }
    int maxPathSum(TreeNode* root) {
        check(root);
        return maxp;
    }
};
```

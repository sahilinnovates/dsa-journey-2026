### 1) Problem: Path with Minimum Effort

**You are given a grid `heights` where each cell represents height. You start at `(0,0)` and need to reach `(n-1, m-1)`. You can move in four directions. The effort of a path is defined as the maximum absolute difference between heights of consecutive cells. Return the minimum effort required.**

**Approach:**

* This problem is a variation of the shortest path problem on a grid.
* Instead of minimizing the sum of weights, we minimize the **maximum edge difference** along the path.
* Applied **Dijkstra’s Algorithm** because the grid can be treated as a graph where each cell is a node.
* Used a priority queue storing `{effort, {row, col}}` so that the state with minimum effort is processed first.
* Maintained a `dist` matrix where `dist[r][c]` stores the minimum effort required to reach that cell.
* For each movement, calculated the absolute height difference.
* The new effort becomes `max(current_effort, edge_difference)`.
* If this new effort is smaller than the stored value, updated it and pushed into the queue.
* The moment the destination cell is popped, its effort is guaranteed to be minimum (Dijkstra property).

**Time Complexity:** `O((n*m) log(n*m))`
**Space Complexity:** `O(n*m)`

**Code:**

```cpp
class Solution {
public:
    int minimumEffortPath(vector<vector<int>>& heights) {
        int n = heights.size();
        int m = heights[0].size();
        vector<vector<int>> dist(n, vector<int>(m,1e9));
        vector<int> row = {-1,0,1,0};
        vector<int> col = {0,1,0,-1};
        priority_queue<pair<int,pair<int,int>>,vector<pair<int,pair<int,int>>>,greater<pair<int,pair<int,int>>>> pq;
        pq.push({0,{0,0}});
        dist[0][0]=0;
        while(!pq.empty()){
            int h = pq.top().first;
            int r = pq.top().second.first;
            int c = pq.top().second.second;
            pq.pop();
            if(r==n-1 && c==m-1) return h;
            for(int i=0; i<4; i++){
                int nr = r+row[i];
                int nc = c+col[i];
                if(nr>=0 && nc>=0 && nr<n && nc<m){
                    int nd = abs(heights[nr][nc]-heights[r][c]);
                    if(max(nd,h)<dist[nr][nc]){
                        dist[nr][nc]=max(nd,h);
                        pq.push({dist[nr][nc],{nr,nc}});
                    }
                }
            }
        }
        return -1;
    }
};
```

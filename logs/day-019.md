### 1) Problem: Dijkstra’s Algorithm (Using Set)

**Given a source node S, find the shortest distance from the source to all vertices in a weighted graph with non-negative weights. If a vertex is unreachable, its distance remains 10^9.**

**Approach:**

* Initialized distance array with large value (1e9) and set source distance to 0.
* Used an ordered set to always extract the node with minimum distance.
* Applied relaxation for all adjacent nodes.
* If a shorter path is found, removed the previous pair from the set and inserted the updated distance.

**Time Complexity:** `O(E log V)`
**Space Complexity:** `O(V)`

**Code:**

```cpp
class Solution{
public:
    vector<int> dijkstra(int V, vector<vector<int>> adj[], int S) {
        vector<int> dist(V,1e9);
        set<pair<int,int>> st;
        st.insert({0,S});
        dist[S]=0;
        while(!st.empty()){
            int nd = st.begin()->second;
            int d = st.begin()->first;
            st.erase(st.begin());
            for(auto it: adj[nd]){
                if(d+it[1]<dist[it[0]]){
                    if(dist[it[0]]!=1e9){
                        st.erase({dist[it[0]],it[0]});
                    }
                    dist[it[0]]=d+it[1];
                    st.insert({dist[it[0]],it[0]});
                }
            }
        }
        return dist;
    }
};
```

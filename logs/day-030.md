### 1) Problem: Find MST Weight using Prim's Algorithm

**short description of problem**
Given a weighted undirected connected graph represented using an adjacency list, compute the total weight of the Minimum Spanning Tree (MST) using Prim’s algorithm.

**Approach:**

* Use a min-priority queue storing (weight, node, parent) to always pick the minimum weight edge.
* Maintain a visited array to avoid reprocessing nodes.
* Add edges contributing to MST and accumulate their weights.
* Traverse adjacency list and push valid neighboring edges into the priority queue.

**Time Complexity:** `O(E log V)`
**Space Complexity:** `O(V + E)`

**Code:**

```cpp
class Solution{
    public:
    int spanningTree(int V, vector<vector<int>> adj[]) {
        priority_queue<vector<int>,vector<vector<int>>,greater<vector<int>>> pq;
        vector<int> vis(V,0);
        // {wt,node,par}
        pq.push({0,0,-1});
        vector<pair<int,int>> mst;
        int sum=0;
        while(!pq.empty()){
            int wt = pq.top()[0];
            int nd = pq.top()[1];
            int par = pq.top()[2];
            pq.pop();
            if(vis[nd]!=0) continue;
            vis[nd]=1;
            if(par!=-1){
                mst.push_back({nd,par});
                sum+=wt;
            }
            for(auto it: adj[nd]){
                if(!vis[it[0]]){
                    pq.push({it[1],it[0],nd});
                }
            }
        }
        return sum;
    }
};
```

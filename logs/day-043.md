### 1) Problem: Vertical Order Traversal

**Given the root of a binary tree, return its vertical order traversal. Nodes are positioned using coordinates where the root is at (0,0), the left child is at (row+1, col-1) and the right child is at (row+1, col+1). If multiple nodes share the same row and column, they must be sorted by value.**

**Approach:**

* Traverse the tree while tracking vertical (`col`) and level (`row`) indices.
* Use a nested ordered structure `map<col, map<row, multiset>>` to maintain sorted order.
* Insert each node value into the corresponding `(col,row)` bucket.
* Iterate through columns from left to right and rows from top to bottom to build the final traversal.

**Time Complexity:** `O(N log N)`
**Space Complexity:** `O(N)`

**Code:**

```cpp
class Solution {
public:
    void inorder(TreeNode* nd, int v, int l, map<int,map<int,multiset<int>>>& mpp){
        if(nd==NULL) return;
        if(nd->left!=NULL) inorder(nd->left,v-1,l+1,mpp);
        mpp[v][l].insert(nd->data);
        if(nd->right!=NULL) inorder(nd->right,v+1,l+1,mpp);
    }
    vector<vector<int>> verticalTraversal(TreeNode* root) {
    	map<int,map<int,multiset<int>>> mpp;
        vector<vector<int>> ans;
        inorder(root,0,0,mpp);
        for(auto it1: mpp){
            vector<int> v;
            for(auto it2: it1.second){
                for(auto i: it2.second){
                    v.push_back(i);
                }
            }
            ans.push_back(v);
        }
        return ans;
    }
};
```

### 1) Problem: Minimum Multiplications to Reach End

**short description of problem**
Given a start value, end value, and an array of multipliers, at each step the current value is multiplied by any element of the array and then taken modulo 100000. Find the minimum number of steps required to reach the end value from the start value. If not possible, return -1.

**Approach:**

* Treat each value (0 to 99999) as a node and perform BFS from the start value.
* For each node, multiply with every element in the array and apply modulo 100000 to generate the next state.
* Use a distance (level) array to track the minimum steps required to reach each value.
* Stop when the target value is reached.

**Code:**

```cpp
class Solution{
public:
    int minimumMultiplications(vector<int> &arr,int start, int end) {
        int n = arr.size();
        vector<int> level(99999,1e9);
        queue<pair<int,int>> q;
        q.push({0,start});
        level[start]=0;
        while(!q.empty()){
            int l = q.front().first;
            int num = q.front().second;
            q.pop();
            if(num==end) return l;
            for(int i=0; i<n; i++){
                int p=(num*arr[i])%100000;
                if(level[p]>l+1){
                    level[p]=l+1;
                    q.push({level[p],p});
                }
            }
        }
        return -1;
    }
};
```
